Battlebots

The Challenge

Build a 3D printed 1-pound battlebot with the goal of defeating the other bot while staying intact.

The Research

This project is pretty straightforward. I’ve built around six different bots for different competitions, and I’m building one right now too. When I first did this competition, I made a simple bot with a sharp, pointy weapon, but I learned that doesn’t work very well. At the end of the day, it’s mostly plastic hitting plastic, so your weapon needs to deliver strong blunt force. One of the coolest parts of this project is the 1-pound weight limit, which forces you to be creative so your bot can hit hard and still stay within weight.

Developing Solutions

This is the project I’m currently working on, so I’ll update this section once it’s finished. So far, I’ve been testing different weapon designs and iterating to find the best option, including whether the weapon performs better spinning sideways or up and down.

Final Project & Reflection

Overall, this is a fun project that’s a great mix of design, engineering, and creativity. I’m still working on the current bot, but I’ve had a lot of fun building different bots for different competitions. Along the way, I’ve learned a lot about what works and what doesn’t in battlebot design.
