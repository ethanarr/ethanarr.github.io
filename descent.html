Safe Descent

The Challenge

I want to design and build a system that can be dropped from any height and orientation, then land safely using thrust vectoring. The plan is to use either servo-based gimbaling or a gyroscope-assisted control system, paired with a propeller.

The Research

So far, I’ve been researching different ways to achieve a safe descent using thrust vectoring. I bought my first microcontroller, a Teensy 4.0, and started researching sensors like the IMU I’m using now.

This project has opened me up to a whole new side of engineering, especially controls engineering, and I’m loving it so far even though it’s challenging. I’m learning about PID control loops and how to implement them, and right now I’m working on code that reads IMU data and uses it to control the thrust-vectoring system.
